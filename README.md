# Python Sorting Algorithms with Memory Limitations - IMDB dataset

Implemented various sorting algorithms to sort the IMDB data set  

Algorithms Implemented
* Quick sort
* Merge sort
* Heap sort
* Selection sort
* Shell sort
* Insertion Sort
